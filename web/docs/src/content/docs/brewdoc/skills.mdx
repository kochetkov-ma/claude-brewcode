# Skills

Skills (recipes) are slash commands that define *what* to do. Brewdoc includes 6 skills focused on documentation sync, memory optimization, Markdown → PDF conversion, content publishing, and an interactive suite guide -- each works independently with no prior setup required.

## Summary table

| # | Skill | Command | Model | Purpose |
|---|-------|---------|-------|---------|
| 1 | auto-sync | `/brewdoc:auto-sync` | opus | Synchronize documentation with code |
| 2 | my-claude | `/brewdoc:my-claude` | opus | Generate documentation about Claude Code installation |
| 3 | memory | `/brewdoc:memory` | opus | Optimize memory files in 4 interactive steps |
| 4 | md-to-pdf | `/brewdoc:md-to-pdf` | sonnet | Convert Markdown to PDF (reportlab / weasyprint) |
| 5 | publish | `/brewdoc:publish` | haiku | Publish to brewpage.app — returns public URL |
| 6 | guide | `/brewdoc:guide` | haiku | Interactive tutorial for the plugin suite |
