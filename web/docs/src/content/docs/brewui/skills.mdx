# Skills

Skills (recipes) are slash commands that define *what* to do. Brewui includes 2 skills
focused on visual and creative tools -- each works independently with no prior setup beyond an API key.

## Summary table

| # | Skill | Command | Model | Purpose |
|---|-------|---------|-------|---------|
| 1 | image-gen | `/brewui:image-gen` | haiku | AI image generation via 5 providers |
| 2 | glm-design-to-code | `/brewui:glm-design-to-code` | opus | GLM vision design-to-code conversion |
