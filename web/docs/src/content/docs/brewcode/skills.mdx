# Skills

Skills (recipes) are slash commands that define *what* to do. Each skill is described in a `SKILL.md` file
with frontmatter metadata and instructions for agents. Brewcode ships two focused skills:
[spec](/brewcode/skills/spec/) for structured task specifications and
[grepai](/brewcode/skills/grepai/) for semantic code search.

## spec

The [spec](/brewcode/skills/spec/) skill turns a rough request into a structured `SPEC.md`. It launches
5-10 research agents in parallel, each analyzing a slice of the codebase (controllers, services, repos,
tests, config, docs), then consolidates their findings -- with deduplication -- into a single
specification. A reviewer validates completeness, consistency, and feasibility before the spec is final.

<Callout type="tip" title="Autonomous mode">
  Pass the <code>-n</code> (--noask) flag to skip the clarifying-questions dialog and run
  the spec end to end without interaction.
</Callout>

## grepai

The [grepai](/brewcode/skills/grepai/) skill manages AI-powered semantic code search backed by Ollama and
the bge-m3 embedding model. It configures `.grepai/`, builds the index, and runs a watch process so
agents can call `grepai_search` to find code by intent instead of literal keywords. The
[grepai configurator](/brewcode/agents/bc-grepai-configurator/) agent generates a config tailored to your
project's languages and layout.
