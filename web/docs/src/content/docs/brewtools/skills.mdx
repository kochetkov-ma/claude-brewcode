# Skills

Skills (recipes) are slash commands that define *what* to do. Brewtools includes 5 skills
focused on text processing, security scanning, SSH management, and deployment -- each works independently with no prior setup required.

## Summary table

| # | Skill | Command | Model | Purpose |
|---|-------|---------|-------|---------|
| 1 | text-optimize | `/brewtools:text-optimize` | sonnet | Text optimization for LLMs |
| 2 | text-human | `/brewtools:text-human` | sonnet | AI artifact removal from code |
| 3 | secrets-scan | `/brewtools:secrets-scan` | sonnet | Secrets and key scanning |
| 4 | ssh | `/brewtools:ssh` | opus | SSH server management |
| 5 | deploy | `/brewtools:deploy` | opus | GitHub Actions deployment, CI/CD |
