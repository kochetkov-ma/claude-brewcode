# Skills

Skills (recipes) are slash commands that define *what* to do. Brewtools includes 3 skills
focused on text processing and security scanning -- each works independently with no prior setup required.

## Summary table

| # | Skill | Command | Model | Purpose |
|---|-------|---------|-------|---------|
| 1 | text-optimize | `/brewtools:text-optimize` | sonnet | Text optimization for LLMs |
| 2 | text-human | `/brewtools:text-human` | sonnet | AI artifact removal from code |
| 3 | secrets-scan | `/brewtools:secrets-scan` | sonnet | Secrets and key scanning |
