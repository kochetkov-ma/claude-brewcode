# Introduction

**Brewcode** is a plugin suite for Claude Code: a **lean spec + search toolkit** plus universal utilities. Write a structured task spec, explore any codebase by meaning with semantic search, then delegate the implementation to a coding agent.

<Callout type="tip" title="Spec, search, delegate">
  `spec` → `grepai` → hand off to the `developer` agent. A tight loop that keeps the agent grounded in your real codebase.
</Callout>

## What makes it different

<CardGrid>
  <Card title="Structured task specs" icon="clipboard" href="/brewcode/skills/spec/">

## What's included

| Plugin | Purpose | Agents |
|--------|---------|--------|
| **brewcode** | Task specs, semantic code search, coding agents | developer, tester, reviewer, architect, creators |
| **brewtools** | Text utilities: prompt optimization, AI artifact removal, secrets scanning, SSH, deploy | text-optimizer, ssh-admin, deploy-admin |
| **brewui** | UI / visual / creative tools (placeholder, empty) | — |
| **brewdoc** | Documentation tools: sync, generation, memory optimization, PDF conversion | bd-auto-sync-processor |

All four plugins install with a single marketplace and load automatically with every Claude Code session.
